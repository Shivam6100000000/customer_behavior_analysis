# 🛒 Customer Shopping Behavior Analysis

## 📌 Overview
This project analyzes customer shopping behavior using a complete data analytics workflow. The goal is to extract meaningful business insights from transactional data and present them through interactive dashboards and presentations.

---

## ⚙️ Workflow
- **Python (Jupyter Notebook)** → Data cleaning & preprocessing  
- **SQL (PostgreSQL)** → Data transformation & analysis  
- **Power BI** → Dashboard & visualization  
- **Gamma AI** → Presentation of insights

---

## 📊 Key Highlights
- Cleaned and structured raw data for analysis  
- Performed business-focused SQL queries  
- Built an interactive Power BI dashboard  
- Presented insights using modern storytelling tools

---

## 📁 Project Structure
├── data/ # Raw & cleaned datasets
├── notebooks/ # Python data cleaning (Jupyter)
├── sql/ # SQL queries for analysis
├── dashboard/ # Power BI file (.pbix)
├── presentation/ # Gamma AI presentation
└── README.md

---

## 🎯 Objective
To transform raw customer data into actionable insights that help understand purchasing behavior, customer segments, and revenue patterns.
